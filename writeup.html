Euler

Change the velocity of the particle based on gravity (and delta_t). Also, change the velocity of the particle based on how far it is away from each attractor (inverse square). The attractor stops attracting once a particle is inside it's radius.

Sphere source

Select a random point uniformally from the surface of a sphere, using the calculated phi value, as described in lecture.

Animated mesh

First we enumerate all of the faces on the mesh, then we select a random face from all of the faces. From here, we linearly interpolate to select a random point from within the given triangle.

Uniform distribution

Instead of selecting a random face, we first create weights for each face based on their respective areas. With these weights, we select a random face using a uniform distribution, and select a random point in the same fashion as before.

Ground plane

Check if the particle is under the plane and reflect and dampen if so.

Axis box

For sink box, we check if the particle is inside the box and kill it if it is. For bounce box we check if the particle is inside the box and which side it's closest too. Then we reflect and dampen according to the closest side.

Sphere collision

Check if the particle is inside the sphere, move it outside and change the velocity.

Spring system

Get the four neighbors for each 'particle' and compute hook's law with each neighbor. This is based on the spring constant and the distance. Then apply those four forces on the velocity of the particle.

Your system

We used the the built in tube module in the THREE library and a sin curve to create sinusoidal tubes. We also have a sink box and a bounce box.

Rendering

We darkened the red and then the green component of the spherical source animation based on the time. This has the particles starting out light and darkening to blue.
